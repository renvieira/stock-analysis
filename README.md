This repository is a learning project where I explored how to connect:

- Backend engineering (FastAPI, services, structured layers)
- Large language models and prompt engineering
- Data from the financial markets domain (both structured and unstructured)

This is not meant to be a production trading tool or a real decision-support system.  
The goal is to learn how to glue software engineering, AI, and data together in a realistic but safe setting.
I DO NOT USE THIS TOOL FOR REAL INVESTMENTS AND/OR TRADING.

I'll be working on different projects in the future for me to use in real life scenarios concerning investments and trading.



- The data and analysis in this project are:
  - Simplified
  - Potentially incomplete or outdated
  - Intended purely for learning about code, APIs, and LLM integration
 


> **Do not use this project to make investment decisions.**  
> Any output generated here should be treated as experimental and educational content.

I have a strong background in financial markets which is why I chose this domain.
